# 🤖 My Copilot Templates

Bộ sưu tập các quy tắc (instructions) tối ưu hóa hiệu suất cho GitHub Copilot và các AI Agents. Được tinh lọc và tùy chỉnh để ép AI viết code chuẩn chỉ, an toàn và đúng cấu trúc.

## 📦 Danh sách Templates

| Công nghệ | File Template | Mô tả |
| :--- | :--- | :--- |
| **Next.js & React** | `templates/nextjs-react.md` | Ép dùng App Router, Strict TypeScript, Tailwind CSS |
| **C# & .NET** | `templates/csharp-dotnet.md` | Clean Architecture, xử lý lỗi, LINQ optimization |
| **Unity** | `templates/unity-gamedev.md` | Tối ưu hóa hiệu suất Update(), quản lý bộ nhớ |

## 🚀 Cách sử dụng

1. Tạo thư mục `.github` ở thư mục gốc của dự án bạn đang làm.
2. Tạo file tên là `copilot-instructions.md` bên trong thư mục đó.
3. Copy nội dung từ file template tương ứng trong repo này và dán vào.
4. Tận hưởng việc GitHub Copilot viết code như một chuyên gia!
